Database Setup:
- in pgAdmim create a new database called StudentDB
- once the database has been createed, go to the query tool and open/paste the contents of the studentDatabase.sql file and execute the query

Download Instructions:
- ensure python is installed by using command 'python --version'
- run the command 'pip install psycopg2' in the terminal to download the python adapter for postgresql
- in the studentDatabase.py file update the password to your password in order connect to your database 

Compilation Instructions:
- use the command 'python studentDatabase.py'
